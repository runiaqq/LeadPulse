# LeadPulse CRM Dashboard

## О проекте

LeadPulse — CRM-lite dashboard для управления лидами и аналитики продаж.

Приложение позволяет:
- отслеживать потенциальных клиентов
- управлять статусами лидов
- анализировать данные продаж
- визуализировать статистику с помощью графиков

---

## Технологии

- React
- Vite
- Tailwind CSS
- ESLint
- Prettier
- Stylelint
- Recharts / Chart.js

---

## Установка проекта
git clone https://github.com/runiaqq/LeadPulse
cd LeadPulse
npm install
npm run dev
